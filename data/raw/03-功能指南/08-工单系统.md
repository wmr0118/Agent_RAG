# 工单系统使用指南

## 工单概述

工单系统用于跟踪和管理客户问题，支持问题分配、流转、升级和统计分析。工单可以来自用户对话、邮件、电话等多种渠道。

## 创建工单

### 从对话创建

在对话窗口中，客服人员可以：

1. 选中用户消息或输入问题描述
2. 点击"创建工单"按钮
3. 系统自动填充问题描述
4. 补充工单信息：
   - 工单标题（必填）
   - 优先级（低/中/高/紧急）
   - 分配给（指定处理人）
   - 标签分类
5. 点击"创建"完成

### 手动创建

在工单管理页面：

1. 点击"新建工单"按钮
2. 填写工单信息：
   - 工单标题（必填，50字以内）
   - 问题描述（必填，详细描述问题）
   - 客户信息（姓名、邮箱、电话）
   - 优先级（默认"中"）
   - 分配给（选择处理人）
   - 标签（选择或创建新标签）
   - 附件（支持图片、文档，最大10MB）
3. 点击"保存"创建工单

### 自动创建规则

可以配置自动创建工单规则：

- 用户对话超过5轮未解决
- 关键词触发（如"投诉"、"退款"）
- 非工作时段用户咨询
- 特定渠道来源（如电话、邮件）

## 工单状态

### 状态类型

工单有以下状态：

- **待处理**：新创建的工单，等待分配处理人
- **处理中**：已分配给处理人，正在处理
- **待回复**：已回复客户，等待客户反馈
- **已解决**：问题已解决，等待确认
- **已关闭**：工单已关闭，不再处理

### 状态流转

标准流转流程：

待处理 → 处理中 → 待回复 → 已解决 → 已关闭

也可以直接关闭或重新打开工单。

### 状态变更通知

状态变更时会自动通知：

- 处理人：工单分配或状态变更时
- 创建人：工单状态更新时
- 客户：工单创建、回复、解决时（如果配置了客户通知）

## 工单分配

### 手动分配

创建工单时可以指定处理人，也可以在工单详情页面修改处理人。

### 自动分配

支持以下自动分配策略：

- **轮询分配**：按顺序分配给在线客服
- **负载均衡**：分配给当前工单数最少的客服
- **技能匹配**：根据工单标签分配给专业客服
- **就近分配**：根据客户地区分配给对应区域的客服

### 工单转派

处理人可以将工单转派给其他客服：

1. 在工单详情页面点击"转派"
2. 选择新的处理人
3. 填写转派原因（可选）
4. 确认转派

转派后原处理人会收到通知。

## 工单升级

### 升级条件

满足以下条件可以升级工单：

- 处理时间超过24小时未解决
- 客户明确要求升级
- 优先级为"紧急"的工单
- 重复创建的相同问题

### 升级流程

1. 系统自动检测或手动触发升级
2. 工单分配给上级主管或专业团队
3. 优先级自动提升
4. 相关人员收到升级通知

## 工单查询

### 基础筛选

支持按以下条件筛选：

- 状态：待处理、处理中、已解决等
- 优先级：低、中、高、紧急
- 标签：选择标签筛选
- 处理人：查看分配给特定客服的工单
- 创建时间：选择时间范围

### 高级搜索

支持全文搜索，可以搜索：

- 工单标题
- 问题描述
- 备注内容
- 客户信息

### 保存筛选条件

常用的筛选条件可以保存为"视图"，方便快速切换。

## 工单统计

### 个人统计

每个客服可以查看：

- 处理的工单总数
- 平均处理时间
- 解决率
- 客户满意度

### 团队统计

管理员可以查看：

- 团队工单总量
- 各状态工单分布
- 平均响应时间
- 工单趋势分析

### 报表导出

支持导出Excel报表，包含：

- 工单列表详情
- 统计数据
- 趋势图表

## 工单模板

### 创建模板

对于常见问题，可以创建工单模板：

1. 在"工单模板"页面点击"新建模板"
2. 填写模板名称和内容
3. 设置默认优先级和标签
4. 保存模板

### 使用模板

创建工单时可以选择模板，自动填充内容，提高效率。

## 最佳实践

1. **及时响应**：新工单应在1小时内分配处理人
2. **详细记录**：问题描述要详细，包含复现步骤
3. **定期跟进**：处理中的工单要定期更新进度
4. **客户沟通**：重要变更及时通知客户
5. **知识沉淀**：已解决的工单可以转为知识库条目
